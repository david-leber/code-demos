# Template Videos Folder

Add your video templates here!

## Quick Start

1. **Record or download short videos (3-10 seconds)**
2. **Save them as .mp4 files in this folder**
3. **Restart the app** to see your templates

## Recording Your Own Templates

### Use Your iPhone:

1. **Set up your shot:**
   - Good lighting (face a window)
   - Plain background
   - Stabilize phone (use tripod or lean against something)

2. **Record short clips:**
   - Open Camera app
   - Video mode
   - Record 3-10 second clips
   - Ideas: wave, dance, nod, thumbs up, funny faces

3. **Transfer to computer:**
   - AirDrop to Mac
   - Email to yourself
   - Use iCloud Photos
   - USB cable

4. **Save to this folder:**
   - Name it something fun: `dance_1.mp4`, `wave_hello.mp4`
   - The filename becomes the template name in the app

## File Naming

The filename determines how it appears in the app:

| Filename | Appears As |
|----------|------------|
| `dance_1.mp4` | Dance 1 |
| `wave_hello.mp4` | Wave Hello |
| `thumbs_up.mp4` | Thumbs Up |
| `funny_face.mp4` | Funny Face |

**Tip:** Use lowercase, underscores instead of spaces.

## Example Template

Here's what a good template looks like:

```
Duration: 5 seconds
Resolution: 1280x720 (720p)
Format: MP4 (H.264)
File size: ~2-5 MB
Person: Centered, facing camera
Face: Clearly visible throughout
Lighting: Even, no harsh shadows
Background: Plain or simple
Action: Waving and smiling
```

## Testing Your Template

After adding a template:

1. Restart the app: Stop server (Ctrl+C), then `python app.py`
2. Open the app in browser
3. Check that your template appears
4. Test by creating a video with it
5. Adjust face position in code if needed (see README.md)
